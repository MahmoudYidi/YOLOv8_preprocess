# 🛠️ YOLOv8 Preprocessing Utilities

This repository provides a suite of preprocessing tools designed to streamline the preparation of datasets using YOLOv8 object detection models. These utilities facilitate tasks such as background removal, image cropping, annotation visualization, and batch processing, ensuring the hyperspectral data is ready for efficient training and inference.
---

## 🔧 Features

- **Background Removal**: Automatically removes backgrounds from images to focus on the object of interest.
- **Image Cropping**: Crops images to specified regions, enhancing model focus on relevant areas.
- **Annotation Visualization**: Visualizes annotations over images to verify dataset accuracy.
- **Batch Processing**: Processes multiple images simultaneously, saving time and effort.
- **Model Inference**: Utilizes a pre-trained YOLOv8 model for object detection tasks.

---

## 🧪 Technologies

- **Programming Language**: Python 3.x
- **Libraries**:
  - `opencv-python`: For image processing tasks
  - `torch`, `torchvision`: For model inference
  - `numpy`: For numerical operations
  - `matplotlib`: For data visualization
  - `PyQt5`: For GUI development
